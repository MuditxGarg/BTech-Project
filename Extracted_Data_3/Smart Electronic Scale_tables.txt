Place weight on Machine
There are two ways of measuring weight
a) Object without container
As shown in the image, if you want to measure the weight of an object directly without a
container, you can proceed as shown
b) Object with container
If you want to measure the object with a container, then put the container on scale then Tare
the scale by pressing Tare button and start measuring the weight
